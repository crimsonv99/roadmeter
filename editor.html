Bulk OSM tag editor — CSV in, changeset out. Locked to a single OSM account.
Connect LIVE OSM

⚠️ Writes go to the live OpenStreetMap (api.openstreetmap.org) — edits affect the real map. Each way in your CSV is written as its own separate changeset. Always Preview first; only adds/changes are written, and each way keeps all its existing tags and nodes.

CSV — way_id + tag columns

First column way_id; every other column header is a tag key (e.g. maxspeed:motorcycle) and the cell is the value to set on that way. Empty cells are skipped. Existing tags and nodes are preserved.
