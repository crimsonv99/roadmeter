# RouteSense

Route navigation-data QA for OpenStreetMap, plus edit-mileage by contributor. Runs entirely in the browser (OSM API + Overpass) — no backend.

**By road** — inspect a route (by relation name/ref/id, or ways sharing a ref) and get a report card: connectivity, oneway consistency, maxspeed sanity, turn restrictions, and ref continuity, with a severity-sorted issue list, map markers, and CSV export.

**By contributor** — edit-mileage per changeset, with an OSMCha-style changeset inspector (map highlight, tag diffs, moved nodes, deleted/old geometry).

Live: https://crimsonv99.github.io/roadmeter/
